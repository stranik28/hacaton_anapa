Documentation to Flask API

Get users by type (/get_users)

To get users by type use param type (0-сотрудник,1 - админ,2-родитель, 3- ребенок)
example: http://hakatonkrasnodar.pythonanywhere.com/get_users?type=1

Search user by fio (/search_user)

To fined user u have to send in param part, part of his fio
example: http://hakatonkrasnodar.pythonanywhere.com/search_user?part=Га

Add user (/add_user)

To add user u have to indicate his type, same as in point 1 in param type
If user type is 0 or 1 u should send fio, photo,inst(Number of his institution), phone.
If user type is 2 u should send fio, photo,inst(Number of his institution), phone, kid (can send many throw space).
If user type is 3 u should send fio,groupa,photo,inst(Number of his institution),phone

example : http://hakatonkrasnodar.pythonanywhere.com/add_user?type=3&fio=Гадя Петрович Хренова&group=12&photo=img/1.png&inst=0&phone=808005553535

Add parents by parents (/send_to_verification_users)

To add parents by parent u should send fio, photo,inst(Number of his institution), phone, kid (can send many throw space).
example: http://hakatonkrasnodar.pythonanywhere.com/send_to_verification_users?type=3&fio=Галина Петровна Корнеплодова&photo=img/3.png&kid=15&phone=8800553535&inst=0

Get users waiting verification (/verificate_user)

example: http://hakatonkrasnodar.pythonanywhere.com/verificate_user

To verificate user (POST) (/verificate_user)

To verificate user use params id,ver(1 to confirm, 2 to reject)
example: http://hakatonkrasnodar.pythonanywhere.com/verificate_user?id=1&ver=1

Happy coding :)
